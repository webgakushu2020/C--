# **02_データの型**

データには「型(type)」と呼ばれる、性質がある  

![データ型](img/02_data1-1.png)

### **数字**
- **int**(イント)型：整数
  - 1, 238, -3 など
- **double**(ダブル)型：小数
  - 0.1, 9.358 など 

### **文字**
- **String**(ストリング)型：文字列
  - こんにちは！, programming, など

### **真偽**
- **bool**(ブール)型：真偽値
  - true(トゥルー)：正しい
  - false(フォルス)：正しくない
    - bool型は「正しい」「正しくない」の **状態** を表す
