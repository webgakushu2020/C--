# **12_メソッド**

処理をひとまとめにしたもの  


### **メソッドを作ろう**

<br>

```dart
using System;

public class Lesson
{
    public static void Main()
    {
        Console.WriteLine("Hello");
        SayHello();
        SayHello();
    }

    public static void SayHello()
    {
        Console.WriteLine("こんにちは");
    }

```

### **引数と戻り値**

<br>

メソッドの中で計算してみよう

```c#

using System;

public class Lesson
{
    public static void Main()
    {
        Sum();
    }

    public static void Sum()
    {
        Console.WriteLine(10 + 20);
    }
}

```

任意の数値を引数で渡して計算してみよう

```c#

using System;

public class Lesson
{
    public static void Main()
    {
        Sum(30);
    }

    public static void Sum(int x) //受け取る
    {
        Console.WriteLine(10 + x);
    }
}


```

複数の引数で渡して計算してみよう

```c#

using System;

public class Lesson
{
    public static void Main()
    {
        Sum(30,20);
    }

    public static void Sum(int x, int y)
    {
        Console.WriteLine(x + y);
    }
}


```

戻り値を返す

```c#

using System;

public class Lesson
{
    public static void Main()
    {
        int num = Sum(30,20);
        Console.WriteLine(num);
    }

    public static int Sum(int x, int y)
    {
        return x + y;
    }
}


```

複数箇所でメソッドを呼び出すこともできる

```c#

using System;

public class Lesson
{
    public static void Main()
    {
        int num = Sum(30,20);
        Console.WriteLine(num);
        int num2 = Sum(130,120);
        Console.WriteLine(num2);
    }

    public static int Sum(int x, int y)
    {
        return x + y;
    }
}


```

変数の有効範囲（スコープ）

```c#

using System;
```

```c#
    public static void Main()
    {
        // Mainメソッドのa変数
```

```c#
using System;

public class Lesson
{
    public static void Main()
    {
        // int num = Sum(10,20);
        int num = 10;
        int num2 = 20;
        
        if(Sum(num,num2)){
            Console.WriteLine("偶数です");
        }else{
            Console.WriteLine("奇数です");
        }
    }

    public static bool Sum(int x, int y)
    {
        int result = x + y;
        if(result % 2 == 0){
            return true;
        }else{
            return false;
        }
    }
}
```
