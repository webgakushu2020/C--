# **09_標準入力**


## **ターミナルからデータを入力し出力してみよう**

### **文字列の入力**

```c#
  var data = "";
  data = Console.ReadLine();
  Console.WriteLine(data);
```

<br>

### **数値の入力**

```c#
  var data = 0;
  data = int.Parse(Console.ReadLine());
  Console.WriteLine(data * 2);
```

<br>

### **複数データの入力**

```c#

var data = "";
data = Console.ReadLine();
Console.WriteLine(data);
data = Console.ReadLine();
Console.WriteLine(data);

```

<br>
